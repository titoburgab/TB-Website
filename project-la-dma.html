A market-by-market sales presentation template built to give an ad sales team a fast, confident story for any DMA — starting with Los Angeles, America's #2 media market.

Built as a repeatable template, so the same structure can be re-skinned market to market without starting from a blank page each time.

Designed and built using Claude for layout, narrative, and content generation, then refined and deployed as a fully interactive site in Figma — an AI-assisted workflow from first draft to finished deck.
